HTML

HTML (HyperText Markup Language) is the backbone of all web pages. It defines the structure of web content.

CSS

CSS (Cascading Style Sheets) is used to style and layout web pages. It makes websites look beautiful and responsive.

JavaScript

JavaScript enables dynamic interactivity on websites. It allows content updates, animations, and much more.

React

React is a front-end library developed by Facebook for building reusable UI components with state management.

Node

Node.js allows developers to write server-side scripts in JavaScript, enabling full-stack JS development.
